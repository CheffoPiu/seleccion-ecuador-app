Ecuador - La Tri

Aplicación móvil con React Native + Expo Go

Materia: Desarrollo de Aplicaciones Móviles
Fecha: Junio 2026

1. Objetivo

Desarrollar una aplicación móvil básica utilizando React Native con Expo Go, con el fin de familiarizarse con el entorno de desarrollo móvil, la estructura de un proyecto Expo y los componentes fundamentales de React Native. La app está dedicada a la Selección Ecuatoriana de Fútbol y se compone de tres pantallas: bienvenida (Splash), inicio (Home) y detalle.

5. Descripción de las pantallas

5.1 Splash Screen

  • Logo oficial de la FEF centrado en pantalla.
  • Fondo azul (#003DA5) y texto amarillo "Ecuador - La Tri".
  • Navegación automática al Home tras 2.5 segundos usando setTimeout dentro de useEffect.

5.2 Home Screen

  • Encabezado con el nombre del equipo.
  • Imagen del escudo de la Selección.
  • Datos básicos: Confederación (CONMEBOL), Entrenador, Estadio (Rodrigo Paz Delgado) y Fundación.
  • Botón interactivo que navega a la pantalla de Detalle.

5.3 Detail Screen

  • Texto descriptivo de la Selección.
  • Botón que muestra un mensaje emergente (Alert).
  • Botón para volver al Home.

6. Lógica de navegación

La navegación se implementó de forma sencilla con el Hook useState en App.js, alternando entre las pantallas mediante una variable de estado. Esto permite entender los fundamentos sin librerías externas. 

const [pantalla, setPantalla] = useState('splash');

{pantalla === 'splash' && <SplashScreen onFinish={() => setPantalla('home')} />}
{pantalla === 'home'   && <HomeScreen onVerDetalle={() => setPantalla('detail')} />}
{pantalla === 'detail' && <DetailScreen onVolver={() => setPantalla('home')} />}

7. Cómo ejecutar la aplicación

npm install      # instalar dependencias
npx expo start   # iniciar el servidor

Luego se escanea el código QR con la app Expo Go en el celular.
